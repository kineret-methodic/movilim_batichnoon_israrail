# מסילות ותכנונים

משחק רצפה פיזי לנאמני תכניות עבודה — רכבת ישראל.

## שלושה תוצרים מסונכרנים
1. לוח רצפה פיזי
2. אתר מנחה מוקרן (`moadilim-poc.html`)
3. תיקי תכנית עבודה במובייל

## הרצה מהירה
פתחו את `moadilim-poc.html` ישירות בדפדפן — אין צורך בשרת.

לתצוגת שגיאות QA: `moadilim-poc.html?devmode=true`

## מקור נתונים
כל התוכן נמצא ב-`src/data/gameContent.json`.
כל טקסט ממשק נמצא תחת `uiCopy` — ניתן לעריכה ללא נגיעה בקוד.
